HTML 语义结构

用 <header> <nav> <main> <article> 这样的标签搭骨架， 而不是一堆没有含义的 <div>。它的用处是： 屏幕阅读器、搜索引擎、浏览器大纲都能读懂页面的层次。

右侧切换两种写法，看「辅助技术眼里的页面」差别有多大。

    Data · 数据层

    05P0

    数据渲染

    把一份数据数组声明式地映射成界面 —— 数据一变，视图自动跟着变，不用手动改 DOM。 它的用处是：列表、卡片、表格都由数据「长出来」，增删即时反映。

    左边动数据（增 / 删），右边的卡片会即时重渲染。这就是 Vue 的响应式。
